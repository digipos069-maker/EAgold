# Gold EA Trading Bot

This is a simple desktop application for a Gold EA trading bot, built with Python and tkinter.

## Features

*   **Real-time Price Display:** Shows the current (simulated) price of Gold.
*   **Manual Trading:** Allows users to place manual "Buy" and "Sell" orders.
*   **Risk Management:** Users can set Take Profit and Stop Loss levels for their trades.
*   **Trading History:** Displays a history of all placed trades.

## How to Run

1.  **Prerequisites:** Make sure you have Python 3 installed on your system.
2.  **Run the application:** Open your terminal, navigate to the project directory, and run the following command:

    ```bash
    python main.py
    ```

## Disclaimer

This is a simplified trading application for demonstration purposes only. The real-time price data is simulated, and the trading functionality is not connected to a live brokerage account. Do not use this application for actual trading.
